# Neko Note 🐱

猫との日常を記録・共有するSNSアプリです。  
猫ごとに投稿を管理できる猫専用SNSです。

---

## アプリ概要

Neko Noteは猫の写真や日常を記録できるSNSです。  
猫ごとにプロフィールを作成し、その猫に紐づいた投稿を作成することができます。

---

## サイトテーマ

猫ごとの成長記録や日常を投稿できるSNS

---

## テーマを選んだ理由

私は猫を2匹、実家でも2匹飼っており、日常の写真をよく撮っています。  
それぞれ性格が違うため、猫ごとに記録を残せるサービスがあれば良いと思い作成しました。

---

## ターゲットユーザー

- 猫を飼っている人
- 猫の成長記録を残したい人
- 猫好きの人

---

## 主な利用シーン

- 猫の日常の投稿
- 成長記録の保存
- 他ユーザーの猫を見る
